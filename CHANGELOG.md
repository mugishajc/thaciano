# Changelog

All notable changes to JCUSSDLib will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-15

### Added
- Initial release of JCUSSDLib
- USSDController for managing USSD operations
- USSDService accessibility service for capturing USSD dialogs
- SplashLoadingService foreground service with overlay widget
- Dual SIM support (Android M+)
- Response keyword matching system
- Loading overlay during USSD operations
- Sequential menu navigation support
- Comprehensive API documentation
- Sample application demonstrating usage
- Apache License 2.0

### Features
- Automated USSD code invocation
- Programmatic response handling
- SIM slot selection for dual SIM devices
- Customizable response parsing
- Timeout handling
- Session state management

### Technical Details
- Minimum SDK: API 23
- Target SDK: API 34
- Language: Java 8
- Build system: Gradle 8.1.0
